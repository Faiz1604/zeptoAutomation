# Zepto

This is a Java Selenium + TestNG project for testing the Zepto e-commerce application.

## Project Structure

- **src/main/java**: Contains page objects, locators, and utility classes
- **src/test/java**: Contains test classes
- **pom.xml**: Maven dependencies and configuration
- **testng.xml**: TestNG suite configuration

## Dependencies

- Selenium WebDriver
- TestNG
- ExtentReports

## Running Tests

Execute tests using TestNG suite:
```bash
mvn test
```

Or run the testng.xml file directly from your IDE.
